# InternLink – Internship 
InternLink is a modern internship tracking and management system designed for students to manage their internship journey in one centralized platform. The system combines internship application tracking with internship progress management features such as tasks, attendance, logbook updates, reports, and analytics.

The current version uses hardcoded data and simulated authentication for demonstration and academic purposes.

# Features included;
## Authentication System
* Login and Register pages
* Hardcoded username and password authentication
* Error message feedback for invalid login attempts
* Redirects users to Dashboard after successful login
## Dashboard
* Internship progress overview
* Quick statistics cards
* Charts and analytics visualization
* Recent activities section
* Responsive modern UI design
## Application Tracking
* Track internship applications
* View application status
* Add new internship applications
* Application details page for current internship company
## Tasks Management
* Add and manage internship tasks
* Task modal popup form
* Track task completion progress
## Attendance & Logbook
* Log internship attendance
* Daily internship logbook entries
* Attendance modal popup form
## Report & Analytics
* Internship progress charts
* Weekly working hours chart
* Productivity overview
## Profile & Settings
* User profile information
* Account settings page
* Theme and UI customization support
